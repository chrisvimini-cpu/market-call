Font Files Required
===================

Place the following CoinDesk Family font files in this directory:

- Family-Regular.woff2
- Family-Bold.woff2

If you don't have these font files, the app will gracefully fall back to:
- Helvetica Neue
- Arial
- System sans-serif

The font files should be in WOFF2 format for optimal performance and browser support.

Note: Font files are referenced in src/styles/fonts.css
